개인정보처리방침

본 개인정보처리방침은 "StudyReminder"(이하 "앱")가 사용자의 개인정보를 어떻게 수집, 이용하고 보호하는지에 대한 내용을 담고 있습니다.

제1조 (수집하는 개인정보 항목 및 접근 권한)

본 앱은 회원가입 절차가 없으며, 서버를 통해 민감한 사용자 개인 식별 정보를 무단 수집하거나 저장하지 않습니다. 기능 제공을 위해 기기 내에서 다음의 권한을 요구할 수 있습니다.

  • 카메라 권한 (선택): 학습 자료나 노트를 촬영하여 AI 퀴즈를 생성하기 위해 단발성으로 사용됩니다.
  • 저장소/갤러리 접근 권한 (선택): 기기에 저장된 학습 이미지를 불러와 AI 퀴즈를 생성하기 위해 단발성으로 사용됩니다.
  • 알림 권한 (선택): 에빙하우스 망각 곡선에 따른 복습 주기 스케줄링 및 타이머 종료 푸시 알람을 제공하기 위해 사용됩니다.
  • 앱 사용 기록 접근 권한 (선택): 사용자가 지정한 '타이머 예외 허용 앱'을 감지하여 타이머를 중단시키지 않기 위한 기기 내부 화이트리스트 검사 용도로만 사용됩니다.

제2조 (개인정보의 수집 및 이용 목적)

수집 단말기 및 접근된 모든 애플리케이션 데이터는 다음과 같은 앱 본연의 기능 제공 목적으로만 일회성 혹은 기기 로컬 내에서 이용됩니다.

  • 사용자가 전송을 명시적으로 요청한 이미지 데이터의 AI 분석 목적 (Google Gemini API 전송)
  • 백그라운드 포그라운드 간 동기화를 위한 시간 측정 및 알림 처리

* 사용자가 입력한 Gemini API 키는 기기의 로컬 암호화 저장소(DataStore/SharedPreferences)에 안전하게 보관되며, 이를 수집하여 외부 백엔드 서버로 전송하지 않습니다.

제3조 (개인정보의 보유 및 이용 기간)

본 앱은 자체 데이터베이스/클라우드 서버에 사용자의 개인 데이터를 보관하지 않습니다. 과목 내역, 사용 시간 통계, 퀴즈 정보, 환경설정 등 앱 구동을 관장하는 데이터는 전적으로 사용자의 모바일 기기 내부(Local Storage)에만 보존됩니다. 앱을 삭제 하거나 데이터(캐시)를 초기화 하실 경우, 기기 내에 저장된 데이터는 완벽히 삭제 복구 불가능합니다.

제4조 (개인정보의 제3자 제공 여부)

원칙적으로 본 앱은 사용자의 동의 없이 개인정보를 외부 업체에 제공하거나 판매하지 않습니다. 단, "AI 퀴즈 생성" 기능 사용 시 사용자가 촬영하거나 첨부한 이미지 파일은 퀴즈 추출 목적으로 사용자가 직접 입력한 API Key 인증 수단을 거쳐 Google Gemini API 인프라로 전송됩니다. 이때 분석 데이터 처리는 Google의 개인정보처리방침을 준용합니다.

제5조 (사용자의 권리 및 행사 방법)

사용자는 언제든지 기기의 휴대폰 설정 장치(예: '설정 > 애플리케이션(앱) > StudyReminder(혹은 회독의 민족)') 메뉴를 거쳐 카메라, 사진 및 미디어, 알림 및 앱 사용 기록 접근 권한 부여를 철회할 수 있습니다. 권한을 철회할 경우 해당 권한을 필요로 하는 확장 기능(AI 퀴즈, 복습 알림 등)의 정상적인 사용이 일부 제한될 수 있으나, 그 외 기본 타이머 및 스톱워치 기능은 정상 제어됩니다.
